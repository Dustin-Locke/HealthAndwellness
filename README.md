# Health and Wellness Tracker

A web application to help users track daily health habits, set achievable goals, and monitor progress over time.

**Course:** `CEN4940` | **Institution:** `Florida State College at Jacksonville` | **Semester:** `Fall 2025`

---

## üöÄ About The Project

The Health and Wellness Tracker provides users with a platform to:
- Track daily health and wellness habits
- Set achievable personal goals  
- Visually monitor progress over time
- Receive reminders to maintain consistency
- Stay motivated through progress insights and streak tracking

This project demonstrates a full-stack application built with Angular frontend and Spring Boot REST API, focusing on core tracking functionality, user authentication, and data visualization.

### Target Users
- Fitness enthusiasts
- People focusing on mental wellness
- Individuals building daily routines

## Project Essentials
If you want to change your postgreSQL password:

Open Command Prompt
- Copy and paste: cd "C:\Program Files\PostgreSQL\17\bin"
- Copy and paste: psql -U postgres -d postgres
- Copy and paste: ALTER USER postgres WITH PASSWORD 'postgres123';
- When creating the database, Be sure to use "CREATE DATABASE healthwellness;"
- Also connect your database in intelliJ

If you want to ignore files on commits

Open Github Desktop app
- Click Repository
- Click Repository Settings
- Click on the "ignore files" on the left pane
- In the right pane, add any files you want to ignore followd by a "/" for everything within its folder

An example would be ".idea/" or ".gitignore/"
